## Tech Stack

**Client:** React, Javascript, HTML, CSS, Google Fonts, Webpack, Babel, ESlint, Jest

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/basicjohn/tap-room.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
